# Welcome screen pictures go here

Place your onboarding images in this folder with these exact names:

- `welcome_1.png` — first welcome page (app introduction)
- `welcome_2.png` — second page (tracing with your finger)
- `welcome_3.png` — third page (getting feedback)

PNG or JPG both work (if you use JPG, update the paths in
`lib/screens/welcome_screen.dart`). Until the files exist, the app
shows placeholder icons instead — it won't crash.

Recommended: roughly square images, at least 600×600 px.
